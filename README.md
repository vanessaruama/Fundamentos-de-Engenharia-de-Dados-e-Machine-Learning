# 🚀 Bootcamp de Engenharia de Dados com Python, ETL e Machine Learning (DIO)

Este repositório reúne todas as atividades, desafios e projetos
desenvolvidos ao longo do bootcamp de **Engenharia de Dados** da
plataforma **DIO (Digital Innovation One)**, com foco em Python, bancos
de dados, ETL, cloud e Machine Learning.

------------------------------------------------------------------------

## 📌 Sobre o Bootcamp

Este bootcamp foi projetado para construir uma base sólida em:

-   🐍 Python para dados
-   🗄️ Bancos de dados (SQL e modelagem)
-   🔄 Processos de ETL (Extract, Transform, Load)
-   ☁️ Fundamentos de Cloud Computing
-   🤖 Machine Learning
-   🧠 IA Generativa e integração com LLMs

Ao longo de **61 horas de conteúdo aplicado**, o programa combina teoria
e prática com foco em problemas reais do mercado de dados.

Além disso, o bootcamp inclui:

-   🎓 Mentorias ao vivo com especialistas
-   💻 Desafios práticos de código
-   📊 3 projetos completos para portfólio

------------------------------------------------------------------------

## 🎯 Objetivo

Desenvolver habilidades práticas para atuar na área de **Engenharia de
Dados**, passando por todo o ciclo:

> Coleta → Tratamento → Transformação → Armazenamento → Análise →
> Geração de Insights

------------------------------------------------------------------------

## 🧩 Atividades do Bootcamp

### 📖 Cursos

Aulas em vídeo curtas e objetivas, focadas em um conteúdo por vez,
permitindo aprendizado no seu próprio ritmo.

-   Explicações práticas
-   Exercícios aplicados
-   Avaliações ao final de cada módulo

------------------------------------------------------------------------

### 🎥 Mentorias (Live)

Sessões ao vivo com especialistas da área, com foco em:

-   Aprofundamento técnico
-   Discussão de casos reais
-   Tira-dúvidas
-   Orientação de carreira

------------------------------------------------------------------------

### 💻 Desafios de Código

Atividades práticas para reforçar o aprendizado:

-   Resolução de problemas reais
-   Desenvolvimento do raciocínio lógico
-   Prática com Python e estruturas de dados

------------------------------------------------------------------------

### 📊 Desafios de Projeto

Projetos completos que simulam desafios do mundo real:

-   Integração de diferentes conhecimentos
-   Construção de pipelines de dados
-   Geração de insights para tomada de decisão
-   Criação de portfólio profissional
